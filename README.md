## Descripción del Proyecto

Este repositorio contiene mis soluciones a varios problemas de LeetCode como parte del curso de Estructuras de Datos.

El objetivo de esta tarea es demostrar el uso y comprensión de diferentes estructuras de datos y algoritmos, tales como arreglos, listas enlazadas, árboles, hashing y técnicas de búsqueda.

Todas las soluciones fueron implementadas en C# y probadas localmente antes de su entrega.

---

## Problemas Resueltos

### Easy

1. **268. Missing Number**

   * Concepto: Arreglos, búsqueda lineal

2. **206. Reverse Linked List**

   * Concepto: Listas ligadas

3. **222. Count Complete Tree Nodes**

   * Concepto: Árboles binarios

4. **771. Jewels and Stones**

   * Concepto: HashSet

5. **136. Single Number**

   * Concepto: Diccionario / conteo

---

### Medium

1. **386. Lexicographical Numbers**

   * Concepto: Recorrido implícito tipo árbol

2. **513. Find Bottom Left Tree Value**

   * Concepto: BFS, uso de cola (Queue)
